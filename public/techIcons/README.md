# Skill icons

Put icon files in this folder: **`public/techIcons/`**

## WordPress & CMS (used by your Skills section)

| Skill         | Icon file       | Format |
|---------------|-----------------|--------|
| WordPress     | `WordPress.svg` | SVG    |
| Custom Themes | `WordPress.svg` | SVG    |

**You need:** one file — **`WordPress.svg`** — in this folder.  
If you prefer PNG, add **`WordPress.png`** and update the `logoPath` in `src/data/repositories/skillsData.js` from `./techIcons/WordPress.svg` to `./techIcons/WordPress.png`.

## Where to get the WordPress logo

- Official: https://wordpress.org/about/logos/  
- Simple Icons (SVG): https://simpleicons.org/?q=wordpress  

Download the SVG or PNG and save it as **`WordPress.svg`** (or **`WordPress.png`**) in **`public/techIcons/`**.
